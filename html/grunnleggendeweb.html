Hva grunnleggende web er

Grunnleggende web er alt som foregår bar skjermen vi ser. For å gjøre det enkelt kan vi dele det opp i tre grupper. Den første gruppen er html struktur. Det kan man tenke på som "skjelettet". Gruppe nummer to er css. Css er som alt av muskler, sener, blod, hud og hår. Nesten som proteiner som settes sammen og danner en viss farge eller størrelse. Den siste gruppen er JavaScript (Js). Js er selve motoren til systemet, eller selve innredningen. Man kan tenke på det som det ekstra du tilføyer til et nakent menneske. JavaScript er det som gjør selve siden interaktiv og klikbar
