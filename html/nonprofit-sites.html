Hva grunnleggende UX/UI er

Grunnleggende UX/UI er alt vi ser og opplever på skjermene våre. Vi kan tenke på UX/UI som et hus, der UX er arkitekturen, hvor dørene i et hus er plassert og alle praktiske valg. Mens UI er intreøret, hvilke farger skal veggene ha, og hvilke dørhåndtak skal dørene ha? UI står for User Interface og handler om det visuelle. Alt vi ser og kan trykke på. Her kommer alle valgene våre inn, hvilke farger, knapper, fonter, ikoner og bildevalg. UX står for User Experience og handler om selve brukeropplevelsen. Alle valgene vi som designere tar og hvordan det blir møtt av brukerne. Er det lett å finne frem? Er fargene godt satt sammen så det er behagelig å se på? Er det et mønster som gjør siden brukervennlig? Og er fonden lett leselig? Det er derfor man utfører ulike tester som lighthouse-testen,
